# Speaky 功能规划

## 🎯 高价值功能

### 1. 多 ASR 引擎支持
- [ ] 添加讯飞语音识别
- [ ] 添加 OpenAI Whisper
- [ ] 添加 Azure Speech
- [ ] 用户可根据网络状况或识别效果切换引擎

### 2. 历史记录
- [ ] 保存最近的识别结果
- [ ] 支持搜索、复制历史文本
- [ ] 可选云端同步

### 3. 快捷短语/模板
- [ ] 预设常用短语，语音触发快速输入
- [ ] 如说"签名"自动输入邮件签名

### 4. 实时字幕浮窗
- [ ] 识别过程中显示悬浮字幕
- [ ] 支持位置和样式自定义

---

## 🔧 实用增强

### 5. 多语言识别
- [ ] 自动检测或手动切换语言
- [ ] 中英文混合识别优化

### 6. 标点智能处理
- [ ] 语音插入标点（如说"逗号"）
- [ ] 自动分段/换行

### 7. 输入目标过滤
- [ ] 白名单/黑名单应用
- [ ] 在特定应用中自动禁用

### 8. 音频质量增强
- [ ] 降噪处理（可考虑 RNNoise）
- [ ] 麦克风选择

---

## ✨ 体验优化

### 9. 识别置信度显示
- [ ] 低置信度词汇高亮提示
- [ ] 方便用户确认修改

### 10. 快捷键增强
- [ ] 多组快捷键配置
- [ ] 不同快捷键对应不同行为（如纯复制、纯输入）

### 11. 统计面板
- [ ] 使用时长、识别字数统计
- [ ] 周/月使用报告
